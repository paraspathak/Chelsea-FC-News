Chelsea FC News

Python Web Scrapper to scrape Chelsea FC news and email then daily.

Download this project as a .zip file

Features of Chelsea FC Scraper

  1. Runs daily at user specified time.

  2. Sends email to users in the database, which can be updated using the text file.

  3. Scrapes weaintgotnohistory.sbnation.com for Chelsea news.

  4. Creates the news along with hyperlink to the story.

Dependency

  1. Python 3 (Link)

  2. Scrapy (Link)

  3. Internet Connection

To run Chelsea FC Web Scraper

Clone this repository and change directory to the cloned folder. Then:

  1. After you are in the main folder of the repository type:

    xdg-open emails.txt

    Then add email addresses of people separated by a new line

  2. If you have all the required dependency then type:

    python controller.py &

    This runs the process in the background and allows you to continue your work
